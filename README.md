## Install and Run

1. Clone the repository:

```bash
git clone https://github.com/prabhatm8000/intern-batch
```

2. Navigate to the project directory:

```bash
cd intern-batch
```

3. Install dependencies:

```bash
npm install
```

4. Run dev server:

```bash
npm run dev
```

5. Open your browser and navigate to http://localhost:3000
